## Recording the Demo

### Screen Recording Setup

**Tools:**
- **Mac:** QuickTime (free), ScreenFlow ($169), OBS (free)
- **Windows:** OBS (free), Camtasia ($250)
- **Browser-based:** Loom (free tier), Tella ($15/mo)

**Settings:**
- Resolution: 1920x1080 minimum
- Frame rate: 30fps
- Audio: External mic (not laptop mic)
- Browser: Hide bookmarks bar, use clean profile

### Recording Tips

1. **Clean up the UI**
   - Use incognito/guest browser profile
   - Hide any personal bookmarks
   - Close other tabs
   - Disable notifications

2. **Script it out**
   - Write every click you'll make
   - Write your voiceover script
   - Practice 2-3 times before recording

3. **Show real interaction**
   - Actually have a voice conversation with the AI
   - Show genuine reactions
   - Don't fake or overdub the AI responses

4. **Capture multiple takes**
   - Record 3-4 full run-throughs
   - Pick the best moments from each
